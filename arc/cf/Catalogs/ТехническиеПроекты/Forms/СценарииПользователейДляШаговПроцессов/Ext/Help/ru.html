Сценарии пользователей для шагов процессов

Форма используется для объединения шагов процессов и сценариев пользователя.
Возможно назначить шагу процесса уже существующий сценарий пользователя или создать новый сценарий пользователя.

Как назначить шагу процесса сценарий работы пользователя?

В таблице нужно заполнить значение в колонке "Сценарий пользователя".

Как назначить шагу процесса сценарий работы пользователя?

В таблице нужно заполнить значение в колонке "Сценарий пользователя".
  • Можно выбрать сценарий пользователя вручную.
  • Команда "Создать и назначить сценарий" позволяет создать новый сценарий пользователя для выделенных строк. Поднадобится указать наименование сценарий пользователя и функцию системы.
  • Команда "Объединить сценарии" позволяет быстро назначить сценарий пользователя нескольким строкам. Для этого нужно выделить одну строку, в которой сценарий уже есть, и несколько строк, для которых надо назначить сценарий.

Для фиксации результата нужно использовать команду "Перенести" или "Перенести и закрыть".
