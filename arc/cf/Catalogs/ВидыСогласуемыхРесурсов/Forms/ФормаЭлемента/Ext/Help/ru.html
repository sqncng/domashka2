Вид планируемого ресурса

Описывает вид ресурса, который необходим для выполнения задачи и по которому потребуется согласование. Например, трудозатраты или деньги.

Как правило, трудозатраты сотрудникам согласуют их непосредственные руководители. Указать согласующего ресурс для сотрудника можно на закладке "Настройки согласования". Согласующего можно указать как для пользователя, так и для группы пользователей.

Для элемента справочника требуется указание:

  • Наименование - имя вида ресурса.
  • Единица измерения - краткое строковое представление.
  • Описание (опционально) - подробное описание назначения вида ресурса.
