Общая задача

Справочник "Общие задачи" предназначен для ведения задач, которые требуется реализовать сразу в нескольких проектах.

Как можно запланировать выполнение общей задачи сразу по нескольким проектам?

Для планирования по проектам можно воспользоваться соответствующей командой "Запланировать по проектам", указав проекты, для которых требуется запланировать общую задача, а также ответственных за выполнение задач по проектам.

При использовании команды список проектов и ответственных заполняется автоматически на основании ранее планировавшихся задач. Список может быть отредактирован вручную пользователем.
