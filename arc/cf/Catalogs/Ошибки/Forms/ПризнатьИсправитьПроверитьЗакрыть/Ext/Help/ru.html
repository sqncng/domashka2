Признать, исправить, проверить, закрыть

Форма предназначена для групповой обработки ошибок. Групповая обработка позволяет выполнить действия с ошибками, установив соответствующие флажки для выполнения требуемых действий. При этом могут выполняться как все возможные действия последовательно, так и только некоторые. Например, можно выполнить только признание ошибок.

Если обрабатывается несколько ошибок, находящихся в разных статусах, для ошибок, не требующих конкретного действия, это действие не будет выполняться. Например, обрабатываются ошибки в статусе Зарегистрирована и Признана. При выборе действий Признать и Отметить исправление зарегистрированные ошибки будут признаны и отмечены как исправленные, а для признанных будет установлена только отметка об исправлении, т.к. они уже были признаны ранее.
