Ошибки

В процессе разработки информационной системы возможно возникновение различных ошибок. Для учета и обработки этих ошибок, публикации информации о них в конфигурации предусмотрен механизм работы с ошибками.

Данная функциональность позволяет регистрировать ошибки, обрабатывать, отслеживать их исправление, управлять версиями исправления.

Форма предназначена для работы с ошибками.
Отфильтровать ошибки по автору, статусу и другим признакам можно при помощи полей быстрых отборов, расположенных в форме.

Для чего предназначена закладка "Ошибки в ветках"?

Одна и та же ошибка может присутствовать в различных ветках разработки (если такие ветки ведутся). На данной закладке отображается список ошибок, присутствующих в различных ветках, в разрезе этих веток. Здесь пользователь может работать с такими ошибками, отбирая при этом, например, только те ошибки, которые требуют исправления.

Как отобрать в списке только те ошибки, которые еще не исправлены?

В форме доступен быстрый отбор по статусам ошибок. Например, для отбора ошибок, подлежащих исправлению, можно установить отбор по статусу Признана.
