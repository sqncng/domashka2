Добавление ИБ для воспроизведения ошибки

Форма используется для добавления информационной базы, требующейся для воспроизведения ошибки, в общий сетевой ресурс, используемый для хранения информационных баз.

В поле Расположение исходной базы следует указать адрес информационной базы, которую требуется приложить. Могут использоваться как файловые, так и клиент-серверные базы.. При этом можно воспользоваться командой Выбрать из списка и выбрать информационную базу из списка информационных баз, зарегистрированных на компьютере пользователя. При выборе из списка адрес информационной базы подставляется в поле Расположение исходной базы автоматически.

В случае, когда исходная база - серверная, в форме доступны дополнительные реквизиты:

В поле База будет скопирована в каталог указывается каталог, расположенный на выделенном ресурсе, куда будет помещена информационная база. Поле заполняется с учетом настроек проекта. В настройках проекта определяется общий каталог расположения информационных баз, прикладываемых к ошибке.

  • Копировать сервером СППР - при установке данного флага на том сервере, под управлением которого работает СППР, будет запущен конфигуратор в пакетном режиме, произведена выгрузка исходной информационной базы в файл выгрузки (dt) и создание файловой копии ИБ.
  • имя пользователя - Имя пользователя исходной информационной базы
  • пароль - Пароль пользователя исходной информационной базы.

Для возможности копирования серверной базы на сервере СППР должен быть установлен конфигуратор соответствующей версии 1С:Предприятия.
