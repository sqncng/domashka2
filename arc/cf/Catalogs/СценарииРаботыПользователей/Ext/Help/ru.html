Сценарии работы пользователей

Сценарии работы пользователей (далее сценарии) используются для описания и выполнения действий пользователя с помощью механизма Автоматизированного тестирования платформы 1С:Предприятие.

Для написания сценариев используется Gherkin подобный язык.

Сценарий может быть создан в текстовом редакторе либо с помощью записи действий пользователя в клиенте тестирования.

Сценарий соотносится с функциями посредством привязки сценария к конкретным функциям.
