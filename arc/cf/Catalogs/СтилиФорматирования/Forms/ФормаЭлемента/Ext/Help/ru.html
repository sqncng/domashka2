Стиль форматирования

Стили форматирования используются для подготовки справки по разрабатываемой конфигурации. При помощи стилей определяется внешний вид справки- шрифт, отступы, выделение.

СППР включает готовый набор предопределенных стилей, которые можно использовать для оформления справки. Кроме того, пользователи могут самостоятельно создавать собственные стили, исходя из своих потребностей.

Для чего служит флаг Стиль-картинка?

Справка может формироваться из абзацев текста и картинок при помощи отдельных элементов справки. Для каждого элемента справки указывается свой стиль форматирования. Если требуется в справке отразить картинку (например, скриншот), который должен располагаться отдельным блоком в результирующем документе справки (не между словами текста), следует использовать стиль Картинка.
