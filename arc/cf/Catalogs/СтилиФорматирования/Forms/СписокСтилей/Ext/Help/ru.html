Стили форматирования

Стили форматирования используются для подготовки справки по разрабатываемой конфигурации. При помощи стилей определяется внешний вид справки- шрифт, отступы, выделение.

СППР включает готовый набор предопределенных стилей, которые можно использовать для оформления справки. Кроме того, пользователи могут самостоятельно создавать собственные стили, исходя из своих потребностей.
