Какую информацию можно и необходимо указать о приоритете?
  • Проект - проект, для которого настраиваются приоритеты. Заполняется автоматически, согласно контексту создания.
  • Наименование - наименование приоритета. Представление, которое будет идентифицировать приоритет в списках, отчетах и командах по изменению приоритета.
  • Порядок - является сквозным для всех проектов информационной базы. Таким образом, для каждого проекта можно будет задать свои наименования приоритетов, но при этом можно будет определить, выше или ниже приоритет одной работы разных проектов.
  • Цвет - позволит выделять колонку с приоритетом заданным цветом в различных формах списков.
  • Описание - подробное описание назначения приоритета.
