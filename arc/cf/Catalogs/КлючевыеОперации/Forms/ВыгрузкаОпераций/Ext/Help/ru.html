Выгрузка ключевых операций

Используется для выгрузки ключевых операций для Центра контроля качества. Открывается при выполнении соответствующих команд из формы списка.

Список

Содержит ровно такой перечень колонок, который используется в выгрузке.

Состав колонок при выгрузке для конфигуратора:

  • Имя – полное имя ключевой операции (без символов шаблона «%»);
  • Целевое время – максимальная нормальная длительность выполнения ключевой операции (утвержденное целевое время отмечено черным цветом, а не утвержденное красным цветом).

Состав колонок при выгрузке для Центра контроля качества:

  • Имя – полное имя ключевой операции (без символов шаблона «%»);
  • Целевое время – максимальная нормальная длительность выполнения ключевой операции (утвержденное целевое время отмечено черным цветом, а не утвержденное красным цветом);
  • Важность – важность от 1 до 9, где 1 максимальная, а 9 минимальная важность;
  • Ответственный – пользователь, ответственный за объект метаданных в который встраивается ключевая операция.

Быстрый отбор

Для списка доступен быстрый отбор по подсистеме. В выгрузку попадают только те ключевые операции, которые относятся к выбранной подсистеме. Чтобы выгрузить все ключевые операции, отбор по подсистеме следует очистить.

Выгрузка, сформированная с отбором по подсистеме, содержит имя этой подсистемы в атрибуте «Items.Description». Имя этой подсистемы в выгрузке может использоваться в алгоритме настройки ключевых операций, чтобы разделить ключевые операции по указанным подсистемам.

Выгрузка

На закладке «Выгрузка» доступен текст, который можно скопировать (для переноса в Центр контроля качества) или сохранить в файл при помощи команды «Выгрузить».

Текст можно редактировать. При выполнении команды «Выгрузить» сохранится именно отредактированный текст. Но нужно помнить, что изменение отбора по подсистеме очищает этот текст.
