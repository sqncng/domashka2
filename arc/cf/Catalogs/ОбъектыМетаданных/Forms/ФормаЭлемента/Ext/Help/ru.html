Объект метаданных

Справочник "Объекты метаданных" содержит информацию о метаданных разрабатываемой конфигурации. Информация о метаданных может загружаться из конфигуратора при помощи обработки "Загрузка метаданных".

Объекты метаданных могут также создаваться в справочнике вручную, на этапе разработки функциональной модели, когда метаданные еще не разрабатываются в конфигураторе.

Для чего служит флаг "Присутствует в основном хранилище"?

При разработке конфигурации, помимо основного хранилища, в котором ведется разработка проекта, могут использоваться отдельные хранилища для разработки технических проектов. При этом в процессе разработки определенные метаданные могут быть реализованы в рамках технического проекта, но еще не помещены в основное хранилище проекта.

Флаг "Присутствует в основном хранилище" показывает, помещен ли объект метаданных в основное хранилище, или нет. Флаг выставляется автоматически при загрузке метаданных из конфигурации.

Для чего служит поле "Ветка-источник"?

В данном поле отображается ссылка на ветку, в которой был создан и загружен объект метаданных. Поле заполняется автоматически при загрузке метаданных из ветки.

Как подготовить справку для объекта метаданных?

Справка готовится для каждого объекта метаданных. Для перехода к редактированию справки объекта метаданных следует использовать команду Элементы справки в панели навигации.

Как заполняется информация о поддержке объектов метаданных?

Если проект использует библиотеки и объект метаданных находится на поддержке у используемой библиотеки, информацию об этом можно указать для объекта метаданных. Нужно установить флаг Стоит на поддержке, выбрать библиотеку, на поддержке которой находится объект, а также указать правило поддержки.

Стоит отметить, что при загрузке метаданных из конфигурации производится также и загрузка информации о поддержке. Для просмотра этой информации можно воспользоваться командой Настройки поддержки в панели навигации.

При использовании нескольких библиотек в проектеодин и тот же объект метаданных может присутствовать в различных библиотеках. Если же объект присутствует только в одной используемой библиотеке, то при загрузке метаданных информация о поддержке для объекта метаданных указывается автоматически.

Что показано на закладке "Встраивание механизмов"?

На закладке "Встраивание механизмов" показаны все механизмы, которые могут быть встроенны в текущий объект метаданных.
Для каждого механизма показан статус его встраивания в объект и состав его функций также со статусом встраивания в объект.
Для выделенных функций доступна групповая установка статуса.
