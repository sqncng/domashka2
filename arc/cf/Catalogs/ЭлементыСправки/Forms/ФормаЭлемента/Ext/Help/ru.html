Элемент справки

СППР позволяет готовить для разрабатываемой конфигурации справку, формируемую в едином стиле по установленным правилам.

Справка для объектов конфигурации компонуется программно из отдельных блоков в соответствии с правилами компоновки. Внешний вид справки: шрифт, отступы, выделение можно задавать непосредственно из программы путем изменения соответствующего стиля в справочнике Стили форматирования.

Текст справки готовится в виде отдельных абзацев справки. Каждый элемент справки соответствует одному абзацу справки.

Справка для подсистемы компонуется:

  • из собственной справки подсистемы;
  • оглавления подчиненных подсистем;
  • оглавления функций, входящих в подсистему;
  • справки функций.

Справка форм объектов метаданных включает следующие составляющие:

  • собственная справка формы;
  • текст справки объекта метаданных (для основных форм объектов). В данный текст включаются элементы справки до первого элемента, имеющего стиль Заголовок

Для остальных объектов справка формируется из собственной справочной информации (подчиненных элементов справки).

Форма служит для редактирования конкретного элемента справки, представляющего собой абзац текста или картинку.

Как используется флаг Включать в справку?

При формировании справки в итоговый текст справки включаются только те элементы справки, для которых установлен флаг Включать в справку. Если элемент справки находится в процессе подготовки, либо по каким-то причинам егь временно не нужно включать в справку, то следует снять флаг Включать в справку.

Для чего используется команда Вставить ссылку?

При подготовке текста справки бывает удобно организовывать перекрестные ссылки в тексте справки. Например, при описании работы с документом, в котором используются данные справочника, можно адресовать пользователя к справке по данному справочнику для получения дополнительной информации.

Для вставки ссылки следует выделить фрагмент текста, который булет использоваться в качестве гиперссылки и выбрать команду Вставить ссылку. При этом выбранный фрагмент текста будет обрамлен в тексте элемента справки специальными тегами, а в таблицу Гиперссылки будет добавлена новая строка. В колонке Текст гиперссылки отражается фрагмент текста справки оформленный как гиперссылка, в в колонке Гиперссылка можно выбрать ссылку на нужный объект метаданных, форму, подсистему. Когда справка будет помещена в конфигурацию, при использовании справки нажатие такой гиперссылки приведет к переходу к справке соответствующего объекта метаданных.

В ряде случаев бывает необходимо организовать ссылки не только на справку других объектов метаданных, нои на какие-то внешние ресурсы, например адрес страницы сайта с полезной информацией. Для реализации такого варианта в строке таблицы Гиперссылки необходимо установить флаг в колонке Текстовая гиперссылка и указать в колонке Гиперссылка собственно требуемую ссылку в виде строки.

Как можно использовать поле Подменяющий элемент справки?

В том случае, когда в различных объектах присутствует однотипный функционал, текст справки может быть одинаковым в разных объектах. При использовании подменяющих элементов справки можно избежать необходимости дублирования такого текста. Для этого можно описать элемент справки в одном из объектов, а в сотальных объектах создать элементы справки, ссылающиеся на подменяющие элементы справки. При формировании справки текст будет компоноваться с учетом подменяющих элементов.
