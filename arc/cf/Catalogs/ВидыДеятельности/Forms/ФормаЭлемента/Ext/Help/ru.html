Вид деятельности

Виды деятельности служат для учета затрат времени на выполнение тех или иных задач в различных разрезах для возможности последующего анализа этих затрат.

Для чего служит флаг Используется?

Флаг использования вида деятельности служит для фильтрации видов деятельности. Если какие-то виды деятельности утратили актуальность, флаг можно снять, при этом такие виды деятельности не будут предлагаться для выбора пользователю при заполнении документа Учет времени.

Как используется вид времени?

Вид времени используется для определения - является ли време, затраченное на работы и отражаемого в учете времени по данному виду деятельности,, рабочм, или не является. Для рабочего времени требуется указание конкретного проекта, в рамках которого выполняются работы, для остального- указание проекта не требуется.

Для чего предназначен флаг Используется в рамках технических проектов?

Необходимо установить данный флаг для тех видов деятельности, которые осуществляются по задачам в рамках технических проектов. Такие виды деятельности не будут доступны для выбора в подситеме "Планирование".
