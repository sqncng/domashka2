Изучение стандартов разработки

Форма изучения стандартов предназначена для пометки выбранных стандартов состоянием Требует изучения:

  • для себя, если форма открыта посредствам вызова команды Изучить;
  • для списка выбранных пользователей, если форма открыта посредствам вызова команды Поручить изучение (дополнительно отображается список с подбором пользователей).

В любом случае можно указать Причину изучения - строковое описание причины изучения, которую увидят выбранные пользователи. А также установить флаг Изучать повторно:

  • Ложь (по умолчанию) - изученные стандарты не будут помечены, как требующие изучения;
  • Истина - даже уже изученные стандарты будут помечены, как требующие изучения.

При выборе групп стандартов, в итоговом списке будут отображены все стандарты, входящие в эти группы.
