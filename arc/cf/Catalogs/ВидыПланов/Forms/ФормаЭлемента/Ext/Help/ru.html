Вид плана

Виды планов позволяют выполнять настройку планирования работ для сотрудников.

Виды планов могут быть двух типов:

  • План занятости - позволяет выполнять длительное планирование в разрезе этапов технических проектов и участников плана
  • Рабочий план - предназначен для оперативного планирования работ на предстоящий период.

Какие данные необходимо указать в виде плана?
  • Наименование - краткое наименование вида плана, которое позволит быстро находить вид плана в списках.
  • Статус - может принимать оно из следующих значений: "Черновик", "Действует", "Не актуален". Планы будут создаваться только для видов планов в статусе "Действует".
  • Описание - подробное описание вида плана.
  • Периодичность - требуется указать количество периодов и вариант периода. Вариант периода может принимать одно из следующих значений: День, Неделя, Квартал, Год.
  • Начиная с - дата с которой будет с заданной периодичностью осуществляться планирование.
  • Владельцы плана - пользователи, которые будут фиксировать план и иметь возможность оставлять комментарии к планируемым работам.
  • Участники планирования - пользователи, по которым будет составляться планы и подводиться итоги их выполнения.
  • Вид согласуемого ресурса - вид ресурса, плановые остатки по которым будут выводиться при составлении плана. В случае, если в информационной базе задан только один вид согласуемого ресурса он будет определен автоматически

Как задать владельцев плана?

Список владельцев плана, которые будут определять настройки плана, фиксировать планы можно задать на закладке "Владельцы плана".
При подборе можно выбирать как отдельного пользователя, так и группы пользователей.

Как задать участников планирования?

Список сотрудников, планы которых будут составляться, можно задать на закладке "Участники планирования".
При подборе можно выбирать как отдельного пользователя, так и группы пользователей.

Можно ли изменить периодичность планирования, если по виду плана уже есть составленные планы?

Да, можно, если новые настройки не противоречат уже составленным планам. Например, возможно изменить период планирования с недели на месяц, если при этом указать новую дату начала действия настроек позднее, чем дата окончания последнего составленного плана.

Для чего требуется указание "Вида согласуемого ресурса" в плане?

"Вид согласуемого ресурса" указывается для отображения остатков запланированных ресурсов по работам в РМ "Планирование" при составлении планов. В случае, если в информационной базе задан только один вид согласуемого ресурса он будет определен автоматически

Для чего указывается "Пороговое значение прочих работ"?

*Данная настройка действует только для рабочих видов планов.

Для удобства отображения в деревья рабочего места "Планирование" работы сотрудника группируются согласно их логической принадлежности. Например задачи одного технического проекта будут сгруппированы вместе, а задачи, не относящиеся к техническим проектам будут сгруппированы в ветке "Непроектные задачи". Для незначительных работ предусмотрена группировка "Прочие работы". В данную ветку будут автоматически попадать виды деятельности и непроектные задачи, у которых плановое время выполнения меньше или равно указанному пороговому значению.

Какие настройки можно задать на закладке "Оформление"

*Данные настройки применимы только для рабочих видов планов.

Можно использовать следующие настройки условного оформления:

  • Цвет текста
  • Цвет фона
  • Текст незаполненного значения

Данные настройки будут централизовано применены ко всем рабочим местам участников вида плана.

Для чего указывается "Пороговое значение для этапа"?

*Данная настройка действует только для рабочих видов планов.

Данная настройка предназначена, для того чтобы ограничить попадание в план занятости этапов технических проектов, по которым запланированы незначительные трудозатраты.

Для этапа технического процесса будет предлагаться указать плановые трудозатраты только в том случае, если для участника плана по этому этапу есть невыполненные задачи и для этих задач:

  • Или план, указанный в задачах суммарно превышает данное значение.
  • Или фактические трудозатраты по данным задачам превышают данное значение.

Как связать вид рабочего плана и вид плана занятости?

*Данные настройка применима только для рабочих видов планов.

Для этого, на форме элемента рабочего вида плана занятости требуется установить флаг - "Составляется на основании плана занятости" и указать план занятости.
Для того чтобы связать вид рабочего плана с видом плана занятости должны быть выполнены следующие условия:

  • Периодичность и количество периодов в настройках видов планов должны совпадать.
  • Даты начала планирования должны быть указаны таким образом, что периоды будущих планов должны совпадать.
