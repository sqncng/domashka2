Подсистема

Справочник "Подсистемы" содержит информацию о подсистемах конфигурации.
Данные в справочник загружаются в процессе загрузки метаданных из конфигуратора при помощи обработки "Загрузка метаданных".
На этапе разработки логической модели подсистемы могут вводиться непосредственно в справочник вручную. При этом заполненный на этапе логического проектирования справочник Функции позволяет достаточно просто выделить подсистемы разрабатываемой конфигурации. В дальнейшем при загрузке метаданных подсистемы, созданные в СППР, синхронизируются с информацией, загруженной из конфигурации.

Можно ли в СППР создавать справку для подсистем?

В СППР возможно создавать справку как для объектов метаданных, так и для подсистем. Для этого должны быть введены элементы справки. Для работы с элементами справки, касающимися подсистемы, необходимо перейти по команде "Элементы справки" в панели навигации формы.
