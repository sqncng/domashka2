Учет времени

Документ предназначен для отражения в учете затрат времени на реализацию задач, работу с ошибками и другую деятельность. Учет ведется в разрезе задач, ошибок, проектов и видов деятельности.

Заполнение документа по данным календаря пользователя

Документ может быть заполнен по данным календаря пользователя. При этом список работ заполняется данными календаря на указанную дату. Для заполнения следует использовать оманду Заполнить по данным календаря.

Подбор ранее выполненных работ

Если пользователь периодически выполняет одну и ту же работу (либо в течение какого-то времени работает над одной и той же задачей), можно использовать команду Подобрать, которая позволяет выбрать ранее выполнявшуюся работу из списка.
