Для чего указывается каталог исполняемого файла 1С:Предприятия?

Каталог исполняемого файла указывается для того, чтобы выполнить запуск 1С:Предприятия требующейся версии в пакетном режиме для создания хранилища. При использовании сервера хранилищ версия запускаемой платформы должна совпадать с версией используемого сервера хранилищ.

Какие действия выполняются системой при создании хранилища?

При создании хранилища система формирует файл поставки на основе конфигурации основного хранилища, создает информационную базу для разработки, загружает в нее файл поставки, создает хранилище технического проекта и подключает базу разработки к хранилищу.

При этом требуется указать каталог размещения базы разработки, имя пользователя базы и его пароль. Если у пользователя имеется база с данными, которые планируется использовать в процессе разработки для целей отладки, можно установить флаг Загрузить базу из файла и указать файл для загрузки.

Создание хранилища выполняется в фоновом режиме. Ход процесса сопровождается комментариями в форме с отметкой о выполнении отдельных этапов процесса создания хранилища.
