Настройки локального запуска сценариев

Форма отвечает за настройки локального запуска сценариев тестирования.

Выбранные настройки будут сохранены только для текущего пользователя СППР.

Секция Менеджер тестирования

Секция Клиент тестирования
  • Адрес ИБ - строка, определяющая базу, которая будет запущена как клиент тестирования
  • Эталонная база - элемент справочника "ЭталонныеБазыТестирования". Из него, на основании профиля пользователя, указанного в сценарии, будет определён пользователь, под которым надо запустить клиент тестирования.
  • Ключи запуска - дополнительные ключи запуска клиент тестирования
  • Тип клиента - принимает значение либо "Тонкий", либо "Web".
  • Менеджер кластера - имеет смысл, когда Тип клиента равен "Web", т.к. когда клиент тестирования запущен в браузере - обмен данными с ним происходит через меджер кластера. Если база опубликована локально - в поле надо указать "localhost".
  • Порт - порт менеджера кластера. Имеет смысл, когда Тип клиента равен "Web". Значение по умолчанию 1541 для серверных баз и 1538 для локальных баз.

Секция Менеджер тестирования
  • Переключатель менеджер тестирования - определяет какой сеанс 1С - менеджер тестирования. Это может быть либо текущий сеанс СППР, либо сеанс другой базы.
  • Адрес ИБ - строка, определяющая базу, на которой будет запущен сеанс, являющийся менеджером тестирования.
  • Каталог программы - путь к установленной версии платформы, на которой будет запущен сеанс менеджер тестирования. Например - "C:\Program Files (x86)\1cv8\8.3.11.2928\bin\"
  • Пользователь - имя пользователя, которое будет использовано для запуска сеанса менеджера тестирования.
  • Пароль - пароль пользователя, который будет использован для запуска сеанса менеджера тестирования.
  • Ключи запуска - дополнительные ключи запуска сеанса менеджера тестирования.
  • Таймаут запуска - количество секунд, в течение которого будет проверяться успешность запуска сеанса менеджер тестирования.
