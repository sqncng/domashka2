Планирование

Обработка "Планирование" предназначена для составления планов работ на период и последующего подведения итогов.
Периодичность планов и участники планирования определяются видом плана.

С чего начать работу?

Выберите вид плана. Страницы обработки заполнятся данными составляемого плана текущего периода и данными последнего зафиксированного плана, если такой по данному виду плана имеется.

Как указать планируемые работы?

Спозиционируйтесь на сотруднике, работы которого вы хотите подобрать.
Страницы подбора, расположенные справа от списка составляемых планов будут заполнены возможны работами выбранного сотрудника.
Доступно четыре страницы подбора:

  • По планам занятости - запланированные над данный период работы в плане занятости.
  • Из предыдущих планов - невыполненные сотрудником работы последнего зафиксированного плана.
  • Ошибки - не закрытые ошибки, которые в данный момент направлены сотруднику.
  • Технические проекты и задачи - активные технические проекты, в которых сотрудник является ответственным или участником, а также не выполненные задачи сотрудника.
  • Виды деятельности - виды деятельности не используемые в технических проектах.

Для выбора работы сотрудника можно использовать:

  • Выбрать двойным кликом - работа будет добавлена в список планируемых работ текущего сотрудника.
  • Использовать перетаскивание - при перетаскивании работы из страницы подбора в таблицу составляемых планов можно добавить выбранную работу и другому сотруднику.

Открыть форму элемента работы на страницах подбора можно используя команду контекстного меню "Показать значение"

Какую информацию система предоставляет для планируемых работ?
  • Наименование - наименование работы.
  • Выполнить к - дата, к которой работу необходимо выполнить. Отображается красным цветом, срок выполнения истек.
  • Статус - текущий статус работы и дата перехода на данный статус.
  • Остаток - Действительно для задач и технических проектов. Выводится информация сколько часов из ранее запланированных на выполнение данной работы еще осталось.

Какие возможности предоставляются по подбору технических проектов и задач?

При подборе технических проектов и задач есть четыре варианта:
Очередь задач - В подборе будут отображаться задачи, согласно выстроенной очереди выполнения задач.
Только технические проекты - В подборе будут отображаться только технические проекты, в которых сотрудник отмечен как участник, или есть по нему невыполненные задачи
Технические проекты и задачи сотрудника - В подборе будут отображаться как технические проекты, в которых сотрудник отмечен как участник, так и невыполненные задачи сотрудника в данных технических проектах без учета иерархии
Технические проекты и все задачи - В подборе будут отображаться как технические проекты, в которых сотрудник отмечен как участник или ответственный, так и все невыполненные задачи этих проектов с учетом иерархии

Какая информация указывается пользователем для планируемых работ?
  • Цель - краткое пояснение целей, которых хочет достичь пользователь по работе в планируемом периоде.
  • Часы (план) - сколько времени планируется в отчетном потратить на данную работу.
  • Комментарий - предназначена для комментариев владельца плана. Колонка не отображается если данные плана открыл сотрудник, не являющийся владельцем плана.

Какие есть дополнительные возможности по заполнению плановых работ?
  • Заполнить видами деятельности по статистике - будут проанализированы три последних аналогичных периода сотрудники и определены виды деятельности, которыми сотрудник занимался хотя бы два из трех периодов. Такие работы будут перенесены план. Среднее время, которое сотрудник тратил на данные виды деятельности будет указано в плане.
  • Отмеченными к переносу работами прошлого периода При подведении итогов зафиксированного плана можно отметить невыполненные работу к переносу в план следующего периода. При выполнении данной команды такие будут перенесены в текущий составляемый план.

Как отметить окончание составления плана?

При окончании составления плана необходимо выполнить команду Зафиксировать план. Такой план считается составленным, в нем нельзя будет изменить состав работ, цели и плановые трудозатраты.

Как изменить период текущего составляемого плана?
  • При помощи гиперссылки изменить - откроется список выбора планов.
  • При помощи кнопок со стрелками - будет выполнено переключение на предыдущий или следующий составляемый план.

Какую информацию систем предоставляет при подведении итогов ранее зафиксированного плана?
  • Наименование - наименование работы.
  • Выполнить к - дата, к которой работу необходимо выполнить. Отображается красным цветом, срок выполнения истек.
  • Статус - текущий статус работы и дата перехода на данный статус.
  • Цель - цель, указанная при составление плана.
  • Часы (план) - ранее запланированные трудозатраты на работу.
  • Часы (факт) - Фактически потраченные на данную работы часы. Данные берутся из подсистемы "Учет времени".

Какая информация указывается пользователем при подведении итогов?
  • Результат - требуется выбрать одно из значений - "Выполнена", "Частично выполнена", "Не выполнялась", "Отменена". Возможно указать значение результата для нескольких работ одновременно. Для этого необходимо выделить несколько строк и установить нужный результат при помощи одной из команд, расположенной в подменю "Результат"
  • Часы (план) - сколько времени планируется в отчетном потратить на данную работу.
  • Комментарий - предназначена для комментариев владельца плана. Колонка не отображается если данные плана открыл сотрудник, не являющийся владельцем плана.

Как при подведении итогов узнать, какими прочими работами занимался сотрудник?


Если в отчетный период были запланированы "прочие работы" или сотрудник занимался внеплановыми работами, то в данных о его работах появится запись "Прочие работы". Если два раза кликнуть по строке, то откроется отчет, который будет содержать подробную информацию о запланированных "прочих работах" и внеплановых работах сотрудника.

Как изменить период текущего зафиксированного плана?

Про помощи гиперссылки изменить - откроется список выбора планов.
Про помощи кнопок со стрелками - будет выполнено переключение на предыдущий или следующий зафиксированный план.

Как изменить состав отображаемых сотрудников зафиксированного плана?


В правом верхнем углу формы располагается гиперссылка с информацией об общем и отображаемом количестве участников планировании. При нажатии на нее откроется форма, в которой можно настроить состав отображаемых участников.
