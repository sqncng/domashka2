Каково назначение плана занятости?
  • Планирование и соблюдение сроков выполнения технических проектов
  • Планирование загрузки сотрудников на длительный период.
  • Построение ориентиров для оперативного планирования сотрудников.

С чего начать работу?
  • Выберите вид плана занятости.
  • Установите, если это необходимо отбор по выводимым участникам плана.
  • Установите дату по которую выводить планы.

Чем отличается информация на разных закладках рабочего места?
  • По сотрудникам - отображается информация с группировкой по подразделениям, сотрудникам и работам (этапы технических проектов и виды деятельности). Предназначена для планирования и сверки загрузки сотрудников в соответствии с нормативами рабочего времени за период.
  • По техническим проектам - отображается информация по с группировкой по техническим проектам, этапам технических проектов и сотрудникам, занятых в реализации этапов. Предзазначена для планирования работ по техническим проектам и обеспечения их выполнения в срок.

Какая информация выводится в план занятости от сотрудников?
  • Участники плана занятости, сгруппированные по подразделениями
  • Этапы запланированных и активных тех. проектов, в которых у сотрудника есть невыполненные задачи. Такие этапы попадают в план автоматически.
  • Непроектные виды деятельности, которыми будет заниматься сотрудник. Их можно добавить в план вручную, или заполнить по статистике.

Для этапов технических проектов:

  • Технический проект этапа и его целевая версия
  • Плановые даты начала и окончания этапа
  • Факт – уже потраченное время на задачи этапа до начала текущего периода планирования
  • Остаток – разница между запланированными в невыполненных на начало планируемого периода задачах этапа трудозатратами и уже потраченным временем на данные задачи
  • Нераспределено – разница между остатком и количеством трудозатрат внесенных в планы занятости на предстоящие периоды
  • Колонки периодов планирования, в которые необходимо внести плановые трудозатраты по этапу или виду деятельности на период. При этом колонки периодов, которые не соответствуют датам, указанным в этапах технических проектов будут выделены красным.

Как установить отбор по отображаемым сотрудникам?


На закладке «По сотрудникам» нажав на гиперссылку, в правом верхнем углу, на которой предоставлена информация о текущем отборе по сотрудникам.

Какая информация выводится в план занятости от технических проектов?
  • Этапы технических проектов сгруппированные по техническим проектам
  • Сотрудники, у которых в данных этапах есть невыполненные задачи.
  • Целевая версия технического проекта.
  • Плановые даты начала и окончания технического проекта или его этапа.
  • Факт – уже потраченное время на задачи этапа до начала текущего периода планирования
  • Остаток – разница между запланированными в невыполненных на начало планируемого периода задачах этапа трудозатратами и уже потраченным временем на данные задачи
  • Нераспределено – разница между остатком и количеством трудозатрат внесенных в планы занятости на предстоящие периоды
  • Колонки периодов планирования, в которые необходимо внести плановые трудозатраты по этапу или виду деятельности на период. При этом колонки периодов, которые не соответствуют датам, указанным в этапах технических проектов будут выделены красным.

Как установить отбор по отображаемым техническим проектам?


На закладке «По техническим проектам» нажав на гиперссылку, в правом верхнем углу, на которой предоставлена информация о текущем отборе.

К чему надо стремиться при заполнении плана занятости?
  • Основная задача составляющего план занятости – добиться распределения трудозатрат участника плана по этапам запланированных технических проектов, с учетом работ по другим видам деятельности и норматива рабочего времени за период
  • Нужно стремиться, чтобы по итогам составления плана занятости колонка нераспределено оказалась пустой, при этом были запланированы другие виды деятельности, а суммарное количество времени за период не превышало норматив рабочего времени для сотрудника на этот период.

Как этапы технических проектов попадают в план занятости?


В план занятости сотрудника этапы попадают автоматически в случае если, на начало текущего периода планирования для данного этапа есть невыполненные задачи нижнего уровня.
При этом, чтобы не замусоривать план занятости мелкими работами, в настройках плана занятости можно задать пороговое значение. Если такое значение задано, то автоматически в план занятости будут попадать только те этапы для которых выполнено одно из условия:

  • Оставшиеся плановые часы по невыполненным задачам больше чем пороговое значение
  • Фактические трудозатраты по невыполненным задачам больше чем пороговое значение

При этом можно добавить этап в план занятости другого сотрудника вручную, даже если для данного сотрудника не запланировано ни одной задачи. Таким образом, поддерживается следующий сценарий:

  • При составлении плана занятости определяется, что при текущей загрузке участник плана не может выполнить запланированные задачи в намеченный срок.
  • Решается, что часть работ может выполнить другой сотрудник

Как добавить вид деятельности в план занятости участника?
  • Вручную при помощи панели подбора, которую можно вывести на экран по команде, расположенной в правом углу панели по сотрудникам.
  • По статистике по команде, расположенной в командной панели и контекстном меню.

Как добавить этап в план занятости участника?
  • Вручную при помощи панели подбора, которую можно вывести на экран по команде, расположенной в правом углу панели по техническим проектам.
  • По команде, которая расположена в командной панели и контекстном меню панели по сотрудникам.

Как быстро распределить трудозатраты?


Рассмотрим следующий сценарий:

  • По этапу у участника плана есть нераспределенное в плане занятости количество часов.
  • При этом решается, что начиная с определенного периода, сотрудник должен заниматься работами по данному этапу определенное одинаковое время за период.
  • Хочется указать количество часов, которое пользователь будет тратить на этап в периоде, и попросить систему указать данное время для последующих периодов, пока нераспределенное время не будет исчерпано.
  • Выполнить данные действия можно при помощи команды «Раскопировать выделенную ячейку вправо (распределить оставшееся время)». При этом:
  • Уже имеющиеся записи по данному этапу в последующих периодах будут удалены, и распределение будет выполнено заново. При этом будут выполнены изменения и в неотображаемых периодах.
  • Для видов деятельности данная команда просто копирует значение из текущей ячейки вправо и только для отображаемых периодов.

Как быстро перейти с одной закладки редактирования плана занятости на другую без потери контекста?


Предположим, что выполняется планирование тех. проекта. Мы находимся на закладке «По техническим проектам» и видим, что для выполнения этапа в срок, нужно запланировать определенным сотрудникам требуемые трудозатраты. При этом сразу же хочется узнать, а какая другая загрузка уже имеется у данного сотрудника в этот период. Не превысим ли мы в итого его нормативное рабочее время на этот период.
Так же актуальна и обратная ситуация. При изменении плановых трудозатрат по этапу сотрудника на закладке «По сотрудникам» возможно захочется узнать как это отразится на выполнении этапа тех. проекта в целом.

Для быстрого перемещения с одной закладки на другую без потери контекста предусмотрены команды:

  • Перейти на другую закладку и добавить в отбор выделенные значения
  • Перейти на другую закладку и установить отбор по выделенным

Для чего предназначена фиксация плана?
  • После составления и согласования планов по всем участникам план на период фиксируется.
  • Плановые трудозатраты зафиксированных планов могут редактировать только владельцы планов.

Как зафиксировать план?


Зафиксировать можно первый отображаемый незафиксированный период. Сделать это можно при помощи команды, расположенной на закладке "По сотрудникам".

Как отменить фиксацию плана?


Отменить можно фиксацию последнего отображаемого зафиксированного периода при помощи команды, расположенной в дополнительном подменю "Еще" формы
