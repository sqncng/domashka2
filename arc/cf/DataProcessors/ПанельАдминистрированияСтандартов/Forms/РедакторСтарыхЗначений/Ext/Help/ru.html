Редактор старых значений

Редактор старых значений предназначен для групповой обработки текстов старого реквизита Стандарт (например, при помощи регулярных выражений). Текстовое поле формы содержит текст, в котором каждая строка соответствует значению старого реквизита Стандарт. Удобно использовать для большого количества ошибок, которым нужно сопоставить новые элементы справочника Стандарты разработки.

Возможный сценарий работы:

  • Из формы сопоставления стандартов открывается эта форма.
  • Значение текстового поля со старыми значениями реквизита Стандарт копируется целиком.
  • Скопированный текст вставляется в текстовый редактор с поддержкой регулярных выражений.
  • При помощи регулярных выражений строки модифицируются так, чтобы остались только коды стандартов.
  • Полученный текст целиком копируется в буфер обмена.
  • Значение текстового поля полностью заменяется на новое значение из буфера обмена:
  • Выполняется команда Записать и закрыть.

ВАЖНО: новое количество строк должно равняться количеству строк до изменения.

Пример

Пример

Текст до изменения:

http://server/standards/123
http://server/standards/456
http://server/standards/789
http://server/standards/1011

Регулярное выражение поиска:

^http:\/\/server\/standards\/(\d+)$

Регулярное выражение замены:

\1

Текст после изменения:

123
456
789
1011
