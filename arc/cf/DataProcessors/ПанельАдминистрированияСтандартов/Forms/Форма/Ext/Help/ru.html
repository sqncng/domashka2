Панель администрирования стандартов

Стандарты разработки можно не настраивать для новой информационной базы СППР без использования внешних стандартов. В остальных случаях необходима описанная ниже настройка.

Настройка

Описанная в этом разделе настройка необходима, только если используется внешняя система для хранения стандартов.

Адрес стандартов задается в виде шаблона с параметром КодСтандарта, который соответствует коду текущего элемента справочника Стандарты разработки. Пример:

http://server.local/standards?C=[КодСтандарта]

Синхронизация

Синхронизацию с внешней системой можно выполнять и до установки указанных выше настроек, но после создания всех используемых при синхронизации дополнительных реквизитов стандартов разработки.

Синхронизацию нужно выполнить до сопоставления старых значений стандартов новым.

Сопоставление старых значений стандартов новым

Если ранее в ошибках были заполнены значения реквизита Стандарт, необходимо выполнить сопоставление старых значений реквизита Стандарт новым значениям, выполнив команду Сопоставить стандарты.
