Справка объектов метаданных

Форма предназначена для работы со справкой объектов метаданных. В списке объектов отображаются объекты метаданных и их формы в соответствии с установленными отборами.

При выборе объекта в правой части формы отображается его справка. Если элементы справки объекта метаданных ведутся для нескольких проектов, то над текстом справки появляется переключатель, с помощью которого можно посмотреть версию справки для выбранного проекта.

Для контроля изменений в справке используются команды:

  • Установить статус "Проверено" - отмечает что справка выбранных объектов проверена;
  • Установить статус "Не проверено" - отмечает что справка выбранных объектов не проверена;
  • Установить статус "Снято с контроля" - отмечает что справку выбранных объектов не требуется контролировать.

Для сравнения текущей версии с проверенной используется команда Сравнить с проверенной версией.
В форме История изменений справки, которая открывается при выборе гиперссылки в колонке Последнее изменение, можно сравнивать любые версии справки.
