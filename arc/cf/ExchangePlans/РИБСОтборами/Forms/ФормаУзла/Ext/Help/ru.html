Наименование узла информационной базы

В данном поле указывается наименование узла распределенной информационной базы. Заданное значение будет отображаться в списке настроенных синхронизаций.

Опциональный состав данных

Позволяет настроить состав данных, которые будут выгружаться в узел распределенной базы. Доступны для выбора следующие данные:

  • Функциональная модель. Управляет выгрузкой справочников Функции, Операции, Объекты данных, Процессы, Шаги процесса, Профили пользователей.
  • Технические проекты. Управляет выгрузкой справочника Технические проекты и связанных с ними данных (Контрольные точки, Согласование техпроектов, Вопросы по техпроектам и т.д.).
  • Требования. Управляет выгрузкой справочника Требования и связанных с ними файлов.
  • Дополнительные отчеты и обработки. Доступно для выбора, если в информационной базе включено использование дополнительных отчетов и обработок.
  • Задачи. Управляет выгрузкой справочника Задачи и связанных с ними файлов.
  • Регламенты. Управляет выгрузкой справочника Регламенты и связанных с ними файлов.
  • Ошибки. Управляет выгрузкой справочника Ошибки и связанных с ними файлов.
  • Целевые задачи. Управляет выгрузкой справочника Целевые задачи и связанных с ними файлов.

Фильтры

Настройка фильтров позволяет ограничить миграцию данных по различным критериям. Заполнение фильтров не является обязательным.

Все доступные фильтры объединены в таблицу. Для настройки фильтра необходимо установить признак использования фильтра, и двойным щелчком в колонке "Состав фильтра" подобрать значения, в соответствии с которыми должны отбираться выгружаемые данные.

Доступны к использованию следующие фильтры:

  • По проектам. Влияет на все выгружаемые данные, в которых так или иначе фигурирует проект (задачи, ошибки, регламенты, техпроекты и многое другое).
  • По разделам проектов. Влияет на выгрузку техпроектов, требований, ошибок, задач, целевых задач и связанных с ними файлов. Не влияет на выгрузку функциональной модели и регламентов.
  • По папкам файлов. Влияет на выгрузку присоединенных файлов.
  • По пользователям. Позволяет скрыть имена пользователей, которые не указаны в списке разрешенных.
  • По группам доступа. Управляет составом выгружаемых групп доступа.

Настройки создания начального образа

Расположены на закладке Прочее. Позволяют ускорить создание начального образа, ограничив выгружаемые версии файлов и версии объектов. Возможны следующие варианты:

  • Выгружать все версии (по умолчанию).
  • Игнорировать все версии (устанавливаются флажки Игнорировать версии...)
  • Игнорировать версии до указанной даты (Устанавливаются флажки Игнорировать версии... , и указывается дата начиная с которой версии должны выгружаться).

При необходимости, невыгруженные версии файлов и версии объектов могут быть выгружены в любое время после создания начального образа. Для этого следует зарегистрировать их с помощью обработки Регистрация изменений для обмена.
