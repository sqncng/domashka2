Замещение пользователей по работе с ошибками

Информация о замещении пользователей используется при работе с ошибками. Механизм работы с ошибками в ряде случаев автоматически направляет ошибку тому или иному пользователю. Например, отработанную ошибку предлагается направить для закрытия пользователю в соответствии с настройками работы с ошибками. Если пользователь временно отсутствует, либо это какой-то служебный пользователь (если ошибка регистрируется при помощи внешних систем с использованием веб-сервиса), настройки замещения позволяют организовать автоматическое перенаправление ошибки другому пользователю в соответствии с указанными настройками.
