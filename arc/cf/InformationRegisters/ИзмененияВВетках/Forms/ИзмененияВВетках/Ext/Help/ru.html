Изменения метаданных

Данный список содержит изменения объектов метаданных, выполненные в соответствующих ветках. Поддерживается функциональность согласования изменений ответственными.

Как можно подробно проанализировать изменения?

В списке можно быстро отобрать изменения по ответственному, статусу согласования, статусу веток, в которых выполнены изменения.

Ответственный может согласовать, или не согласовать изменения, воспользовавшись соответствующими командами для установки требуемого статуса согласования.

Согласующий может при необходимости заменить себя другим согласующим, воспользовавшись командой назначения согласующего в контекстном меню.
