Изучаемые стандарты

Форма истории изучения позволяет ознакомиться с хронологией изменения состояния стандартов для пользователей. Открывать эту форму напрямую большого смысла не имеет, эта форма автоматически открывается с правильными отборами при выполнении команды История изучения из формы списка стандартов разработки.

Отборы Пользователь и Стандарт позволяют сузить перечень изменений до одного стандарта для одного пользователя и отобразить понятный список со следующими колонками:

  • Стандарт - стандарт, для которого отслеживаются изменения;
  • Пользователь - пользователь, для которого отслеживаются изменения;
  • Дата - дата изменения состояния или причины изучения стандарта;
  • Состояние - состояние стандарта на указанную дату;
  • Причина изучения - причина изучения стандарта.
